# based-writing
collection of configs for based writing

set-up on Linux Mint

tools used:
* zellij (terminal multiplexer)
* vim with plugins: startify, vimtex (text editor)
* zathura (pdf viewer)
* ranger (file manager)
* neomutt + mutt-wizard (email)
* sc-im (spreadsheets)
* vimiv (image viewer)
* vieb (web browser)
